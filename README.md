# 厦门市第九中学 - 智能教案生成系统

## 项目概述

厦门市第九中学智能教案生成系统是一个基于Web的智能应用，专门为中学教师设计，用于快速生成高质量的主题班会教案。系统结合了本地模板和DeepSeek AI API，能够根据教师输入的基本信息智能生成结构完整、内容丰富的教案内容。

## 核心功能

### 🎯 智能教案生成
- **AI驱动生成**：集成DeepSeek API，基于先进的大语言模型技术生成专业教案
- **本地模板备用**：当API不可用时，自动使用预设的本地模板生成教案
- **实时预览**：输入信息后立即在预览区域显示效果

### 📋 完整教案结构
系统生成的教案包含以下标准结构：
- **基本信息**：时间、班级、班主任、主持人、主题
- **教育目标**：3个具体可衡量的教学目标
- **过程内容**：详细的班会流程和时间安排
- **收获反思**：班会后的总结反思和教育效果评估

### 💾 便捷导出功能
- **HTML导出**：一键导出完整的HTML格式教案文件
- **内容复制**：快速复制教案文本内容到剪贴板
- **自动命名**：导出文件自动按"教案_班级_主题_日期.html"格式命名

## 安装与配置

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 网络连接（用于API调用）

### 快速开始
1. **下载项目文件**：将所有文件保存到本地目录
2. **打开应用**：双击 `index.html` 文件在浏览器中打开
3. **开始使用**：无需额外安装，直接输入信息即可生成教案

### API配置（可选）
如需使用AI生成功能，需要配置DeepSeek API密钥：

1. **获取API密钥**：
   - 访问 [DeepSeek官网](https://deepseek.com)
   - 注册账号并申请API访问权限
   - 创建API密钥

2. **配置密钥**：
   - 在应用界面找到"DeepSeek API密钥"输入框
   - 粘贴您的API密钥
   - 点击"智能生成教案"即可使用AI功能

> **注意**：API调用可能会产生费用，请查看DeepSeek官方定价。如果没有API密钥，系统会自动使用本地模板。

## 使用方法

### 基本操作流程

1. **填写基本信息**：
   - 选择班会时间
   - 输入班级名称（如：初一(1)班）
   - 填写班主任和主持人姓名
   - 输入班会主题（如：安全教育、心理健康教育等）

2. **API密钥配置**（可选）：
   - 如有DeepSeek API密钥，在相应输入框中填写
   - 如无密钥，系统将自动使用本地模板

3. **生成教案**：
   - 点击"智能生成教案"按钮
   - 等待系统处理（约10-30秒）
   - 查看右侧预览区域的生成结果

4. **导出使用**：
   - 点击"复制内容"复制文本格式教案
   - 点击"导出HTML教案"下载完整HTML文件
   - 使用下载的教案文件进行教学

### 界面说明

- **左侧输入区**：填写教案基本信息
- **中间操作区**：生成按钮和操作选项
- **右侧预览区**：实时显示教案内容和生成结果

## 常见问题解答（FAQ）

### ❓ API相关问题

**Q: 没有API密钥可以使用吗？**
A: 可以。系统内置了本地模板，当没有API密钥或API调用失败时，会自动使用本地模板生成教案。

**Q: API调用出现401错误怎么办？**
A: 401错误表示API密钥无效。请检查：
- API密钥是否正确复制粘贴
- 密钥是否已过期或被撤销
- 账户是否有足够的额度

**Q: 遇到429错误是什么意思？**
A: 429错误表示请求频率过高。请等待几分钟后再试，或联系DeepSeek调整API限制。

### ❓ 功能使用问题

**Q: 生成的内容不满意怎么办？**
A: 您可以：
- 调整输入的主题描述，使其更具体
- 重新点击生成按钮尝试不同的结果
- 手动编辑导出的HTML文件进行微调

**Q: 导出的HTML文件如何打开？**
A: 导出的HTML文件可以用任何现代浏览器打开，也可以直接打印使用。

**Q: 系统支持哪些主题？**
A: 系统支持所有常见的班会主题，包括但不限于：
- 安全教育（交通安全、网络安全等）
- 心理健康教育
- 学习方法指导
- 班级管理
- 德育教育

### ❓ 技术问题

**Q: 页面显示不正常怎么办？**
A: 请尝试：
- 刷新页面
- 清除浏览器缓存
- 使用最新版本的浏览器

**Q: 系统需要联网吗？**
A: 如果需要使用AI生成功能，需要联网调用API。如果只使用本地模板，可以离线使用。
